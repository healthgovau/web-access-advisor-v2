Web Access Advisor Test Suite

Welcome to the Accessibility Testing Platform

This test suite contains intentionally created accessibility defects designed to benchmark and validate accessibility testing tools.

🎯 Comprehensive Testing

Test against 100+ accessibility defects across 5 different pages, covering WCAG 2.1 Level A and AA guidelines.

🔧 Multiple Tools

Compare results from Lighthouse, axe-core, NVDA screen reader, and your custom WAA tool.

📊 Benchmarking

Track detection rates and identify areas for improvement in your accessibility testing tools.

Instructions: Navigate through the test pages using the sidebar menu. Each page contains specific types of accessibility defects. Use the accessibility-defects.md file to track your testing results.

🧪 Testing Guidelines

  • Static Analysis: Run Lighthouse, axe-core, and your WAA tool on each page
  • Interactive Testing: Some forms are meant to be filled out and submitted, others should only be analyzed statically
  • Screen Reader Testing: Use NVDA to navigate through all content and interactions
  • Form Instructions: Each page clearly indicates which forms to interact with and which to leave alone
